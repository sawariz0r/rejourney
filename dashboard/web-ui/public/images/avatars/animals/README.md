Animal avatar SVGs are sourced from Twitter Twemoji.

Source: https://github.com/twitter/twemoji
Asset path: https://github.com/twitter/twemoji/tree/master/assets/svg
Graphics license: CC-BY-4.0, per the Twemoji LICENSE-GRAPHICS file.

Only the animal SVGs referenced by the app are kept in this folder.

Some Rejourney anonymous animal names do not have exact Unicode emoji artwork,
so those names use the closest available Twemoji animal while preserving the
anonymous animal name in the UI:

- alpaca: llama
- capybara: beaver
- chinchilla: mouse face
- fawn: deer
- ferret: otter
- lemur: monkey face
- meerkat: badger
- quokka: kangaroo
- squirrel: chipmunk
- wombat: koala
