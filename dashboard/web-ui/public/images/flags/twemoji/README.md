The Iran regional flag asset (`1f1ee-1f1f7.svg`) is intentionally replaced
with the current `Flag of Iran.svg` from Wikimedia Commons, based on the
official ISIRI standard and marked public domain by Wikimedia Commons.

This folder keeps the regional flag SVGs used by dynamic dashboard country
codes so flags render consistently without relying on native emoji fonts.
