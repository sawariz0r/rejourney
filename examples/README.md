# Rejourney Examples

This folder contains example applications demonstrating how to integrate the Rejourney SDKs.

Web examples:

- `web-next`: Next.js App Router with `@rejourneyco/browser/next`
- `web-sveltekit`: SvelteKit with `@rejourneyco/browser/svelte`
- `web-nuxt`: Nuxt client plugin with `@rejourneyco/browser/nuxt`

React Native examples:

- `react-native-boilerplate`
- `react-native-bare`
- `brew-coffee-labs`
