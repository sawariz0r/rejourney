## Fitur

### Capture Pixel Perfect
Playback video FPS nyata yang menangkap setiap pixel, termasuk Mapbox (Metal), shader kustom, dan view yang dipercepat GPU.

### Live Incident Stream
![Issues feed](https://rejourney.co/images/issues-feed.png)

Lihat crash, error, dan rage tap secara real-time dengan laporan crash instan.

### Deteksi Error/ANR/Crash
![ANR issues](https://rejourney.co/images/anr-issues.png)

Deteksi otomatis untuk ANR dengan thread dump lengkap dan analisis main thread.

### Journey Mapping
![User journeys](https://rejourney.co/images/user-journeys.png)

Visualisasikan navigasi pengguna, temukan titik drop-off, dan optimalkan funnel konversi.

### Heatmap Interaksi
![Heatmaps](https://rejourney.co/images/heatmaps.png)

Lihat lokasi tap, swipe, dan scroll untuk memperbaiki penempatan UI.

### Stabilitas Global
![Geo intelligence](https://rejourney.co/images/geo-intelligence.png)

Pantau performa dan stabilitas di berbagai region sebelum masalah berdampak ke pengguna.

### Growth Engines
![Growth engines](https://rejourney.co/images/growth-engines.png)

Lacak retensi, segmen loyalitas, dan dampak release terhadap pengguna aktif.
