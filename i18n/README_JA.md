## 機能

### Pixel Perfect Capture
Mapbox (Metal)、カスタムシェーダー、GPU アクセラレーションされた view を含め、レンダリングされたすべての pixel を real FPS で記録します。

### Live Incident Stream
![Issues feed](https://rejourney.co/images/issues-feed.png)

クラッシュ、エラー、rage tap をリアルタイムに確認し、即座にクラッシュレポートを取得できます。

### Error/ANR/Crash Detection
![ANR issues](https://rejourney.co/images/anr-issues.png)

完全な thread dump と main thread analysis による ANR の自動検知。

### Journey Mapping
![User journeys](https://rejourney.co/images/user-journeys.png)

ユーザーがアプリ内をどのように移動しているかを可視化し、離脱ポイントと funnel を改善できます。

### Interaction Heatmaps
![Heatmaps](https://rejourney.co/images/heatmaps.png)

tap、swipe、scroll の位置を把握し、UI 配置を最適化します。

### Global Stability
![Geo intelligence](https://rejourney.co/images/geo-intelligence.png)

地域別の performance と stability を監視し、問題がユーザーに届く前に把握できます。

### Growth Engines
![Growth engines](https://rejourney.co/images/growth-engines.png)

retention、loyalty segment、release の影響を追跡します。
