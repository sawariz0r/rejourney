## 기능

### Pixel Perfect Capture
Mapbox (Metal), 커스텀 셰이더, GPU 가속 view까지 렌더링된 모든 pixel을 실제 FPS로 캡처합니다.

### Live Incident Stream
![Issues feed](https://rejourney.co/images/issues-feed.png)

크래시, 에러, rage tap을 실시간으로 확인하고 즉시 crash report를 받을 수 있습니다.

### Error/ANR/Crash Detection
![ANR issues](https://rejourney.co/images/anr-issues.png)

전체 thread dump와 main thread analysis를 포함한 자동 ANR 감지.

### Journey Mapping
![User journeys](https://rejourney.co/images/user-journeys.png)

사용자가 앱에서 이동하는 방식을 시각화하고, 이탈 지점과 funnel을 개선합니다.

### Interaction Heatmaps
![Heatmaps](https://rejourney.co/images/heatmaps.png)

사용자의 tap, swipe, scroll 위치를 보고 UI 배치를 최적화합니다.

### Global Stability
![Geo intelligence](https://rejourney.co/images/geo-intelligence.png)

지역별 performance와 stability를 모니터링해 문제가 사용자에게 영향을 주기 전에 발견합니다.

### Growth Engines
![Growth engines](https://rejourney.co/images/growth-engines.png)

retention, loyalty segment, release impact를 추적합니다.
